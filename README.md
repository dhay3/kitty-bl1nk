![bl1nk](./bl1nk.png)

### Automatic Installation

1. Run  `kitten themes`
2. Search for `bl1nk` and press `Enter`
3. Reload kitty

### Manual Installation

1. Download the `bl1nk.conf` from this repository
2. Place `bl1nk.conf` in `~/.config/kitty/`
3. Add `include bl1nk.conf` to your `kitty.conf` file
4. Reload kitty
